
# VISADERM GRAFTSCAN™

Sistema avançado de análise de unidades foliculares baseado em inteligência artificial.

## 🔬 Sobre o projeto
O VISADERM GRAFTSCAN™ é uma plataforma desenvolvida para:
- análise de enxertos capilares
- contagem de unidades foliculares (UF)
- contagem de fios por UF
- avaliação de integridade dos enxertos
- suporte à decisão cirúrgica

## 🚀 Funcionalidades
- Upload de imagem
- Análise automatizada (VISADERM V2.1)
- Visaderm Quality Score™
- Graft Integrity Index
- Risk Score cirúrgico
- Dashboard em tempo real

## 🧠 Diferenciais
- Multi-Hair Axis Detection
- White Hair Detection
- Estrutura baseada em evidência cirúrgica
- Interface otimizada para iPhone

## ⚙️ Tecnologias
- HTML / CSS / JavaScript
- OpenCV (pipeline futuro)
- IA supervisionada (versão futura)

## 📲 Como usar
1. Acesse:
   https://SEU-LINK-AQUI.github.io/graftscan/
2. Faça upload da imagem
3. Clique em "Analisar"
4. Visualize os dados em tempo real
